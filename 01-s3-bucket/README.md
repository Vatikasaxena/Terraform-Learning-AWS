# Terraform S3 Bucket

## Project Overview

This project creates an AWS S3 bucket using Terraform.

## Resources Created

- AWS S3 Bucket

## Terraform Concepts Used

- Provider
- Resource
- State File
- Output

## Terraform Commands Used

terraform init

terraform plan

terraform apply

terraform output

terraform destroy

## Description

Terraform AWS provider is used to communicate with AWS APIs.
The S3 bucket is created using the aws_s3_bucket resource.
Terraform state file maintains the mapping between Terraform configuration and the actual AWS infrastructure.
